🔍 Local OCR Lab

A web app that extracts text from scanned PDFs using four open-source OCR engines — all running locally inside a Docker container with no external APIs and no paid services. Built to compare engine accuracy, measure performance, and export clean text from scanned documents.

↗ Open Live App GitHub →

What it does

  • Upload a scanned PDF and select any page to process
  • Control image quality with a DPI selector (72 to 300)
  • Run OCR with any of four engines — or all at once for side-by-side comparison
  • Visualize detected words and regions with bounding box overlays
  • Filter low-confidence detections with a confidence threshold slider — no re-running needed
  • Separate timing shown for model load vs. OCR inference
  • Download extracted text as a .txt file
  • Jaccard word overlap matrix between engines (no reference text required)
  • CER and WER accuracy metrics when reference text is provided
