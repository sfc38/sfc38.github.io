💬 WhatsApp Chat Analyzer

An interactive web app that turns your WhatsApp chat export into a full analysis report — message statistics, word trends, emoji breakdowns, word clouds, and activity patterns. Supports both English and Turkish text.

↗ Open Live App GitHub →

What it does

  • Parses WhatsApp chat exports (.txt) and structures them into a clean dataset
  • Computes per-user message counts, word counts, and activity streaks
  • Plots message volume over time, by hour of day, and by day of week
  • Identifies top words per sender using NLP lemmatization and stopword removal
  • Generates a group word cloud from the most frequent terms
  • Tracks media sharing — images, videos, audio, stickers, and URLs
  • Extracts and counts emojis per user
  • Supports English (spaCy en_core_web_sm) and Turkish (tr_floret_web_lg) text
  • Downloadable HTML report with all charts and stats embedded

Tech Stack

Python Streamlit spaCy pandas Plotly WordCloud emoji regex NumPy Matplotlib
